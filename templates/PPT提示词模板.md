# PPT提示词模板

## 1. 课堂汇报大纲

```text
我要做一个【时长】分钟的课堂汇报，主题是【主题】，听众是【听众背景】。请帮我设计【页数】页 PPT 大纲，每页包括标题、核心观点、建议材料和演讲重点。页面文字要简洁。
```

## 2. 读书报告结构

```text
我需要汇报一本书/一篇文章：【书名或文章名】。请帮我设计 PPT 结构，包括作者背景、核心问题、主要观点、论证路径、我的评价和讨论问题。请控制在【页数】页以内。
```

## 3. 项目展示

```text
我要展示一个项目：【项目简介】。请按“问题、目标、方案、过程、结果、反思、下一步”的逻辑生成 PPT 大纲，并给出每页适合放图表、流程图还是案例。
```

## 4. 标题优化

```text
下面是我的 PPT 页标题：【粘贴标题列表】。请帮我改得更清楚、更适合展示。要求标题能表达观点，不要太夸张，每个标题不超过 18 个字。
```

## 5. 演讲备注

```text
下面是我的 PPT 大纲：【粘贴大纲】。请为每一页写 80-120 字演讲备注，语气自然，适合课堂展示。不要增加未经核查的数据或案例。
```
