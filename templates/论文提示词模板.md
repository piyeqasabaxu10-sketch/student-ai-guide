# 论文提示词模板

## 1. 选题发散

```text
我是【专业/年级】学生，课程名称是【课程名】，论文要求是【字数/主题范围/格式要求】。我对【兴趣方向】感兴趣。请帮我列出 10 个可进一步缩小的论文选题方向，并说明每个方向适合从哪些材料入手。请不要替我写论文，只做选题辅助。
```

## 2. 题目缩小

```text
我现在的题目是【题目】。请帮我判断它是否过大，并从研究对象、时间范围、案例范围、核心概念和研究问题五个角度给出缩小建议。输出 5 个更具体的题目备选。
```

## 3. 提纲设计

```text
请基于这个论文题目【题目】设计一个课程论文提纲。要求包括：引言、问题背景、核心概念、分析部分、结论。请说明每一部分应该回答什么问题，并提醒哪些地方需要我自己查找文献或数据。
```

## 4. 文献阅读整理

```text
下面是一篇文献的摘要或笔记：【粘贴内容】。请帮我整理为：研究问题、研究方法、核心观点、可引用的思路、与我的题目【题目】的关系。不要编造文献信息。
```

## 5. 论证检查

```text
下面是我的论文段落：【粘贴段落】。请帮我检查论证是否清楚，指出跳跃、重复、概念不明或证据不足的地方。请只给修改建议，不要直接替我生成最终版本。
```
