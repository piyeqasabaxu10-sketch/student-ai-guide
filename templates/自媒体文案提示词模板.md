# 自媒体文案提示词模板

## 1. 选题拆解

```text
我的账号方向是【方向】，目标受众是【人群】。请围绕【主题】拆出 20 个真实、有讨论价值的选题，并按学习型、经验型、观点型、清单型分类。不要写夸张标题。
```

## 2. 脚本大纲

```text
请为选题【选题】设计一个短视频脚本大纲，时长约【时长】秒。结构包括开头问题、主要观点、例子、转折和结尾提醒。语气自然，不要过度煽动。
```

## 3. 标题改写

```text
下面是我的标题：【标题】。请给出 10 个更清楚、更具体的标题版本，要求真实克制，不制造焦虑，不使用夸张承诺。
```

## 4. 文案润色

```text
下面是我的文案草稿：【粘贴内容】。请帮我改得更顺、更像真人表达，保留我的观点，不要加入未经证实的案例或数据。
```

## 5. 复盘分析

```text
这是我发布后记录的数据和反馈：【粘贴数据/评论观察】。请帮我分析可能有效的地方、观众困惑的地方和下一条内容可以改进的方向。不要把相关性当成因果关系。
```
