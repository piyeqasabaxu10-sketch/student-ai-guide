# 考试复习提示词模板

## 1. 制定复习计划

```text
我将在【时间】后参加【考试/课程】考试，当前基础是【基础情况】，每天可用时间是【时间】，薄弱部分是【薄弱点】。请帮我制定一个可执行的复习计划，并保留复盘和休息时间。
```

## 2. 知识点解释

```text
请用适合【年级/基础】学生理解的方式解释这个知识点：【知识点】。请包括核心定义、常见误区、一个简单例子和两个自测问题。
```

## 3. 错题分析

```text
下面是我的错题和错误答案：【粘贴内容】。请帮我分析错误原因，归类到概念不清、审题问题、计算问题、记忆问题或方法问题，并给出复习建议。
```

## 4. 章节总结

```text
请根据以下课堂笔记整理复习提纲：【粘贴笔记】。输出包括：核心概念、重点公式/观点、可能考法、易错点和复习优先级。
```

## 5. 自测题生成

```text
请围绕【章节/主题】生成 10 道自测题，难度从基础到中等。先只给题目，不要给答案。等我回答后，再帮我批改并解释。
```
