# 简历优化提示词模板

## 1. 经历改写

```text
下面是我的真实经历：【粘贴经历】。目标岗位是【岗位】。请帮我把经历改写成简历项目符号，突出具体任务、方法和结果。不要添加我没有做过的内容。
```

## 2. 岗位匹配

```text
这是目标岗位描述：【粘贴 JD】。这是我的简历内容：【粘贴简历】。请帮我分析哪些经历最匹配，哪些表达需要调整，哪些内容可以弱化。请给出修改理由。
```

## 3. 空泛表达检查

```text
请检查下面这些简历句子是否空泛：【粘贴内容】。请指出问题，并改写成更具体、可解释、适合面试追问的表达。不要夸大成果。
```

## 4. 项目经历梳理

```text
我做过一个项目：【项目背景、我的任务、过程、结果】。请帮我整理成简历项目经历，包含项目简介、我的职责、关键行动和可说明的成果。
```

## 5. 面试追问模拟

```text
下面是我的一段简历经历：【粘贴内容】。请模拟面试官提出 8 个追问，并帮我判断哪些地方需要提前准备事实、数据或案例。
```
