# 办公总结提示词模板

## 1. 会议纪要

```text
下面是会议记录：【粘贴内容】。请整理成会议纪要，包括会议主题、关键结论、待办事项、负责人、截止时间和待确认问题。不要补充记录中没有的信息。
```

## 2. 周报整理

```text
下面是我本周做过的事情：【粘贴列表】。请帮我整理成周报，分为本周完成、进行中、遇到的问题、下周计划。语气简洁，不夸大成果。
```

## 3. 邮件草稿

```text
我需要给【对象】写一封邮件，目的为【目的】，背景是【背景】。请帮我起草一版清楚、礼貌、不过度客套的邮件，并列出需要我确认的信息。
```

## 4. 方案摘要

```text
下面是一份方案草稿：【粘贴内容】。请帮我压缩成一页摘要，包括背景、目标、核心方案、资源需求、风险和下一步。
```

## 5. 复盘总结

```text
下面是一个项目的过程记录：【粘贴内容】。请帮我整理复盘，分为目标、实际结果、做得好的地方、问题原因、经验沉淀和后续动作。
```
